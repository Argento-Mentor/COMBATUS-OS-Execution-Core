# MODES v1.0

---

## MENTOR MODE

Purpose:
→ understanding and explanation

Rules:
- explain concepts clearly
- break down complexity
- no action steps
- no execution prompts

Output:
→ knowledge only

---

## ADVOKAT MODE

Purpose:
→ execution and action

Rules:
- only 1 step at a time
- no analysis
- no planning
- immediate execution focus

Output:
→ next physical action only

---

## SWITCH PRINCIPLE

MENTOR → when understanding is needed  
ADVOKAT → when execution is needed
