# COMBATUS OS — USE CASES

## 1. INDIVIDUAL DECISION MAKING
Use case: You are stuck in overthinking.

Process:
- define problem
- reduce to 1 action
- execute immediately

---

## 2. TASK EXECUTION
Use case: Large task feels complex.

Process:
- break into smallest unit
- identify first physical action
- execute ≤2 minutes

---

## 3. PROBLEM SOLVING
Use case: No clear solution path exists.

Process:
- remove noise
- isolate core constraint
- act on smallest controllable variable

---

## 4. PRIORITY CONFLICT
Use case: multiple competing tasks.

Process:
- select 1 irreversible impact task
- park all others
- execute first step only

---

## 5. COGNITIVE OVERLOAD
Use case: too many thoughts, no clarity.

Process:
- stop expansion
- compress to 1 sentence problem
- convert into action

---

## 6. SYSTEM RECOVERY
Use case: breakdown or stagnation.

Process:
- reset to micro-step
- ignore long-term planning
- re-enter execution loop
